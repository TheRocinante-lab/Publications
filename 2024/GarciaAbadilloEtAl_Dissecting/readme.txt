* GENO_mapping.csv: Table with 3 columns (ID, CHROM, POS) containing the physical position of each SNP marker.
* GENO_marker_matrix.csv: Table containing genotypes IDs in the rows and Marker IDs in the columns.
* GENO_original_pedigree.csv: Table with 4 columns (ID, Family, P1, P2) containing the pedigree for each genotype. 
* PHENO_raw_data.xlsx contains the phenotypic information for breeding lines ("segregantes") and varieties ("diversity panel").
